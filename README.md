# Personalized-Webpage

## Description
This is a simple personalized webpage to practice content on js introduction, the project is a webpage built using HTML, Tailwind CSS, CSS, and JavaScript.

## Features
- User enters name and age
- Stores data using localStorage
- Displays a personalized greeting
- Calculates age in months
- Checks if the user is an adult
- Displays a motivational quote five times

## Technologies Used
- HTML
- Tailwind CSS
- CSS
- JavaScript
- localStorage
